Districts of India

A comprehensive directory of all administrative districts in India, indexed with the official Local Government Directory (LGD) codes provided by the Ministry of Panchayati Raj.

Schema Definition

4 Fields Defined
Field Name Type Description Sample Value
District Name STRING The official name of the district in English. "Bangalore Urban"
LGD Code key INTEGER Primary Key. Unique ID assigned by the LGD system. 553
State Name STRING Name of the state or UT the district belongs to. "Karnataka"
Establishment Year INTEGER The year the administrative district was formed. 1986

Python (Pandas) 
import pandas as pd

url = "https://api.opendata.in/v2/geo/districts.parquet"
df = pd.read_parquet(url)

# Filter by state
karnataka_districts = df[df['state_name'] == 'Karnataka']
print(karnataka_districts.head())
cURL (REST API) 
curl -X GET "https://api.opendata.in/v2/geo/districts?lgd_code=553" \
     -H "Accept: application/json" \
     -H "Authorization: Bearer YOUR_API_KEY"
